# 🧠 Calculadora de Concordância

## Kappa de Cohen vs Gwet’s AC1

---

## 📌 Descrição

Esta é uma **calculadora online interativa** desenvolvida para auxiliar estudantes e pesquisadores na análise de **concordância entre avaliadores**.

A ferramenta calcula:

* **Concordância observada (Po)**
* **Kappa de Cohen**
* **Gwet’s AC1**
* **Intervalo de confiança de 95% (IC95%)** para ambas as métricas

Além disso, apresenta o **passo a passo detalhado dos cálculos**, que pode ser ativado sob demanda.

---

## 🎯 Objetivo

Fornecer um recurso didático e aplicado para:

1. Ensino de metodologia científica
2. Interpretação de concordância em estudos clínicos
3. Apoio à elaboração de TCC, artigos e pesquisas

## ⚙️ Funcionalidades

* Entrada simples de dados (tabela 2x2)
* Cálculo automático em tempo real
* Exibição de **IC95%**
* Botão para **mostrar/ocultar cálculos detalhados**
* **Modo claro e escuro**
* Otimização para **dispositivos móveis**
* Botão para **copiar resultados**
* Geração de **link compartilhável** com parâmetros

---

## 📊 Fórmulas utilizadas

### Kappa de Cohen

κ = (Po − Pe) / (1 − Pe)

### Gwet’s AC1

AC1 = (Po − Pe*) / (1 − Pe*)

### Intervalo de Confiança (aproximação normal)

IC95% = estimativa ± 1,96 × erro padrão

---

## ⚠️ Limitações

* O cálculo do intervalo de confiança utiliza **aproximação normal**
* Não substitui métodos robustos para publicação científica
* Implementação limitada a **duas categorias e dois avaliadores**

---

## 🚀 Como usar

1. Inserir os valores da tabela (a, b, c, d)
2. Clicar em **Calcular**
3. Visualizar os resultados
4. Opcional: clicar em **Mostrar cálculos** para detalhamento
